Farmako+ — Medical Appointment & Pharmacy Assistant

Farmako+ is a smart healthcare assistant designed to help users easily book medical appointments, manage prescriptions, and access pharmacy services. The system streamlines doctor availability tracking, appointment scheduling, and medicine lookup through a clean, user-friendly interface.

  • Doctor availability dashboard with real-time updates
  • Appointment booking system with patient scheduling workflow
  • Pharmacy module for checking medicine stock and alternatives
  • Integrated user authentication and profile management
  • Built with React / React Native UI and SQL-based backend
